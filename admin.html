        Einstellungen

        Admin-Passwort ändern

        Das Admin-Passwort gilt nur für diesen Browser. Zum Ändern wird das aktuelle Passwort benötigt.

        Automatischer Selbsttest

        Der Selbsttest prüft die wichtigsten Seiten in zufälligen Abständen. Erst wenn ein Problem wiederholt auftritt, wird ein Admin-Alarm ausgelöst.

        Wenn das aktuelle Passwort nicht mehr bekannt ist, können die lokalen Admin-Zugangsdaten in diesem Browser zurückgesetzt werden. Danach ist eine neue Ersteinrichtung erforderlich.

        API-Diagnose (PostgreSQL-Sync)

        Prüft ob die Railway-API erreichbar ist, Registrierung und Login funktionieren und Nutzer korrekt in PostgreSQL gespeichert werden. Das Testkonto wird nach dem Test automatisch entfernt.
